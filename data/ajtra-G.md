## 8. Usage of uints/ints smaller than 32 Bytes (256 bits) incurs overhead
### Description
When using elements that are smaller than 32 bytes, your contract's gas usage may be higher. 
This is because the EVM operates on 32 bytes at a time. Therefore, if the element is smaller than that, 
the EVM must use more operations in order to reduce the size of the element from 32 bytes to the desired size.
Use a larger size then downcast where needed

## 9. Initialize variables with default values are not needed
### Description
If a variable is not set/initialized, it is assumed to have the default value (0 for uint, false for bool, address(0) for address,etc). 
Explicitly initializing it with its default value is an anti-pattern and wastes gas.
